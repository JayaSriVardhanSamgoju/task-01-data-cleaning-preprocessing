## Data Preprocessing Performed

### Missing Value Handling

* **Age**: Imputed missing values group-wise based on the passenger's class (`Pclass`) and gender (`Sex`) to preserve granular demographic distributions.
* **Embarked**: Imputed missing values with the mode of the dataset ('S').
* **Cabin**: Replaced null values with `'Unknown'` and engineered a binary column `HasCabin` (1/0) indicating whether a cabin was assigned.

### Duplicate Removal

* Checked for duplicate passenger rows using `df.duplicated().sum()`.
* Dropped duplicates programmatically to ensure data hygiene.

### Categorical Encoding

Applied one-hot encoding using `pd.get_dummies` with `drop_first=True` to nominal features:

* Sex
* Embarked
* Title (extracted from passenger names)

### Feature Scaling

Scaled numerical features (`Age`, `Fare`, `FamilySize`) using `StandardScaler` (Z-score normalization) fitted on the training split to prevent data leakage.

### Outlier Detection

Detected outliers using the **Interquartile Range (IQR)** method and capped values at the lower and upper bounds to preserve raw records while limiting skewness.

### 2. Boxplot of Numerical Features (Before Capping)

![Boxplot After Capping](images/boxplot_after.png)
* **What it shows**: Outlier boxplots after applying Interquartile Range (IQR) capping. Fares above `65.63` and Ages above `57.75` have been capped to stabilize the distributions without deleting records.
